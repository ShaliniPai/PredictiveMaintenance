# Predictive Maintenance MLOps Project

This project develops a machine learning model to predict whether an engine requires maintenance using sensor data such as engine RPM, oil pressure, fuel pressure, and temperature readings.

## Project Workflow
1. Data Preparation and Cleaning
2. Exploratory Data Analysis
3. Model Building and Experimentation
4. Model Deployment using Docker and Streamlit
5. CI/CD Automation using GitHub Actions
6. Deployment to Hugging Face Spaces

## Technologies Used
- Python
- Scikit-learn
- Streamlit
- Docker
- Hugging Face Hub
- GitHub Actions

## Model
A Random Forest model was trained and tuned to predict engine maintenance requirements.

## Deployment
The model is deployed using a Docker container and Streamlit interface hosted on Hugging Face Spaces.
